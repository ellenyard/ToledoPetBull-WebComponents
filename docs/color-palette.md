# Site Color Palette

## Primary Colors
- **Accent Color:** #7AC142 (logo green)
- **Links:** #5A9B32 (darker green)
- **Headings:** #222222 (dark charcoal)
- **Body Text:** #333333 (charcoal gray)

## Neutral Colors
- **Site Background:** #F6F1E7 (warm off-white)
- **Borders:** #E0E0E0 (light gray)

## Interactive States
- **Normal Links:** #5A9B32 (darker green)
- **Hover Links:** #7AC142 (logo green - brighter)
- **Primary Button:** #5A9B32 background
- **Primary Button Hover:** #7AC142 background
