# Toledo's Pet Bull Project - Website Components

Custom code and components for toledospetbullproject.com

## What This Repository Contains

- **child-theme/** - Astra child theme with performance optimizations and custom CSS
- **page-components/** - Full page HTML/CSS (saved as we update pages)
- **reusable-components/** - Buttons, cards, modals - reusable anywhere
- **docs/** - Documentation and notes

## Site Information

- **Platform:** WordPress with Astra theme
- **Hosting:** GoDaddy
- **CDN:** Cloudflare
- **Caching:** WP Rocket
- **Colors:** See docs/color-palette.md

## Workflow

1. Edit files locally in this repository
2. Test changes
3. Commit to Git: `git add . && git commit -m "Description"`
4. Push to GitHub: `git push`
5. Upload changed files to WordPress server
